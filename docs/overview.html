NSM Footnotes for ExpressionEngine 2.0 allows you to add references to your content while editing and output them as footnotes in your template. Name and reuse the same references throughout the document multiple times. Customise the footnote output just the way you want with flexible tags.

Hows it work

Adding references

Adding references to your content is very simple. Wrap double brackets ([[…]]) around any text or html that you would like to become a reference in your footnotes and NSM Footnotes will store this as a footnote for rendering later in the {footnotes} tag-pair.

Reusing references

References can be named by adding a hash(#) and a reference identifier when the same reference is used multiple times throughout a document. Example:

The first reference: [[ #ref1 Footnote content ]]
can be reused later like so: [[ #ref1 ]]

Output references as footnotes

NSM Footnotes has a flexible tag pair for out putting the footnotes. There's plenty of options to check out in the tag reference.

Requirements

Technical requirements include:

  • ExpressionEngine 2
  • PHP 5.2+

Installation

  1. Download the latest version of NSM Footnotes.
  2. Extract the .zip file to your desktop
  3. Copy system/expressionengine/third_party/nsm_footnotes to system/expressionengine/third_party

Sample templates

The download includes sample templates to kick start your implementation.

  1. Copy samples/nsm_footnotes.group to your template directory
  2. Syncronise the templates
  3. Browse to http://yoursite.com/index.php?nsm_footnotes
