NSM Footnotes for ExpressionEngine 2.0 allows you to add references to your content while editing and output them as footnotes in your template. Name and reuse the same references throughout the document multiple times. Customise the footnote output just the way you want with flexible tags.

Hows it work

Adding references

NSM Footnotes requires references to be added inline using a simple {ref} tag pair. Content inside of the tag pair will become the footnote content and the tags will be replaced with an anchor linking to the generated footnote.

Reusing references

References can be named using the name="" parameter when the same reference is used multiple times throught a document. Example:

The first reference: {ref name="ref1" } Footnote content {/ref}
can be reused later like so: {ref name="ref1" /}

To reuse the reference use a single {ref} tag with a name="" parameter and a closing slash; Just like a single HTML tag.

Output references as footnotes

NSM Footnotes has a flexible tag pair for out putting the footnotes. There's plenty of options to checkout in the tag reference.

Requirements

Technical requirements include:

  • ExpressionEngine 2
  • PHP 5.2+

Installation

  1. Download the latest version of NSM Footnotes.
  2. Extract the .zip file to your desktop
  3. Copy system/expressionengine/third_party/nsm_footnotes to system/expressionengine/third_party

Sample templates

The download includes sample templates to kick start your implementation.

  1. Copy samples/nsm_footnotes.group to your template directory
  2. Syncronise the templates
  3. Browse to http://yoursite.com/index.php?nsm_footnotes
