Smartdown / Markdown compatibility

If you are using NSM Footnotes with a custom field that is processed by Smartdown or Markdown you will need to adjust your tag parameters to accommodate this.

The left_delimiter and right_delimiter tag parameters allow you to change the delimiters used by NSM Footnotes to define resources.

By default the delimiters are [[ and ]]. Changing these to different values will allow you to combine NSM Footnotes with Smartdown / Markdown text.

{exp:nsm_footnotes left_delimiter="%%" right_delimiter="%%"}
